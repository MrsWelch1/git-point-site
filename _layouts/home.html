What is GitPoint?

GitPoint is an open source GitHub client with a minimally beautiful UI. Built with React Native, it is the most complete (100% free) iOS client and is the easiest way to use GitHub on your device.

Why GitPoint?

Every single unofficial client that I tried to user were either simple, single-feature applications or required payment for even the most basic of requirements (such as accessing a private repository or commenting on an issue).

I was suprised to not be able to find a single client that had everything I needed, so I built this bad boy.

Does this have everything?

Nope. Well, not yet. There are definitely features that still need to be included (plus you may very well notice a few bugs here and there right now).

Please take a look at the issue list to see all of the features we plan on adding. Please don't hesitate to file an issue for a bug or a feature request!

Can I contribute?

Yes please! Please feel free to put up an issue or PR for any issue or feature request. Even if you have little to no experience with React Native, I'll be more than happy to help :).

Take a look at CONRIBUTING.md for all the steps you need to get the project running locally as well how you can contribute.
