Control your notifications

View and control all of your unread and participating notifications.

Repository Details

GitPoint is an open source GitHub client with a minimally beautiful UI. Built with React Native, it is the most complete (100% free) iOS client and is the easiest way to use GitHub on your device.

Complete issue and pull request

Communicate on issue/pull request conversations, apply labels, assignees and more. With GitPoint, you can even review and merge pull requests with a design you're already familiar with.

Another GitHub client?

Every single iOS client that I tried to use was either too simple (single-feature applications) or required payment for even the most basic of requirements (such as accessing a private repository or commenting on an issue).

I was pretty suprised to not be able to find a single client that had everything I needed, so I built this bad boy.

Y no Android?!?

Since this was built with React Native, publishing an Android version is something that we definitely need to do.

There's an open issue to track this, so take a look there to get an idea of when it'll be released.

Does this have everything?

Nope! Well, not yet :). There are definitely features that still need to be included and you'll most probably notice a few bugs here and there.

Please take a look at the issue list to see all of the features we plan on adding. Please don't hesitate to file an issue for a bug or a feature request!

Can I contribute?

Yes please! GitPoint is open source - that's what makes it so great. Please feel free to put up an issue for anything you notice or a pull request to fix an existing one. Even if you have little to no experience with React Native, I'll be more than happy to help :).

Take a look at the contributing guidelines for all the steps you need to get the project running locally as well as detailed steps on how you can contribute.
